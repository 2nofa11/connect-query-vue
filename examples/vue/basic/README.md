# connect-query-vue basic example

A basic example of using [connect-query-vue](https://github.com/2nofa11/connect-query-vue) with Vue 3 and TanStack Query.

## Use as a template

```bash
vp create github:2nofa11/connect-query-vue/examples/vue/basic
```

## Development

```bash
vp install
vp dev
```

## Test

```bash
vp test
```
